## Proyecto práctico de Reinforcement Learning

### Integrantes:

- Luis Sanchez Zamora
- Edgar A. Valdés A.
- Víctor Cano Prados

### Descripción:

El objetivo de este proyecto es implementar un agente de Aprendizaje por Refuerzo Profundo basado
en el algoritmo Deep Q-Network (DQN), capaz de aprender a jugar al entorno Space Invaders de
Atari.
Se emplea el entorno ALE/SpaceInvaders-v5 de Gymnasium y una red neuronal convolucional que
procesa directamente los frames del juego.
El criterio de éxito del proyecto es alcanzar una recompensa media superior a 20 puntos en fase de
test.
